Line Plot - Weekly Temperature¶

This graph shows temperature changes during the week. I chose a line plot because it is useful for showing trends over time.

In [33]:
from matplotlib.typing import MarkerType
days = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
temp = [18, 20, 19, 22, 23, 24, 21]

plt.figure(figsize=(8,5))
plt.plot(days, temp,
         marker="o",
         linestyle="--",
         color="blue",
         label="Temperature",)

plt.title("Weekly Temperature")
plt.xlabel("Days")
plt.ylabel("Temperature (°C)")
plt.legend()
plt.grid(True)

plt.savefig("imgs/line_plot.png", dpi=300, bbox_inches="tight")
plt.show()
